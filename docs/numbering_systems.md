## Numbering Systems

### Decimal (base 10) &mdash; uses numbers 0-9

| 10<sup>2</sup> | 10<sup>1</sup> | 10<sup>0</sup> | 
| :----:         | :----:         | :----:         |
| **1**          | **2**          | **8**          |

| Calculation        | Result        |
| ----               | ----          |
| 8 x 10<sup>0</sup> | 8             |
| 2 x 10<sup>1</sup> | 20 (2x10)     |
| 1 x 10<sup>2</sup> | 100 (1x10x10) |
| Total              | 128           |

### Binary (base 2) &mdash; uses numbers 0 and 1

| 2<sup>3</sup> | 2<sup>2</sup>  | 2<sup>1</sup> | 2<sup>0</sup> | 
| :----:        | :----:         | :----:        | :----:        |
| **1**         | **0**          | **0**         | **1**         |

| Calculation        | Result      |
| ----               | ----        |
| 1 x 2<sup>0</sup>  | 1           |
| 0 x 2<sup>1</sup>  | 0           |
| 0 x 2<sup>2</sup>  | 0           |
| 1 x 2<sup>3</sup>  | 8 (1x2x2x2) |
| Total              | 9           |

**1001** binary = **9** decimal

### Hexadecimal (base 16) &mdash; uses numbers 0-9 and letters A-F

| 16<sup>2</sup> | 16<sup>1</sup> | 16<sup>0</sup> | 
| :----:         | :----:         | :----:         |
| **2**          | **3**          | **F**          |

| Calculation        | Result        |
| ----               | ----          |
| F x 16<sup>0</sup> | 15            |
| 3 x 16<sup>1</sup> | 48 (3x16)     |
| 2 x 16<sup>2</sup> | 512 (2x16x16) |
| Total              | 575           |

 **23F** hexadecimal = **575** decimal
